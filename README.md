# MyFundedTournament (MFT)

Forex trading competition platform. Traders pay USDT entry fees, compete on MT5 demo accounts, highest % gain wins a real funded trading account.

## Tech Stack
- Frontend: Next.js 14 + CSS Variables → Vercel
- Backend: Node.js + Express → Railway
- Database: PostgreSQL → Supabase
- Payments: NOWPayments (USDT TRC-20)
- Trading data: MetaApi (read-only MT5 sync)
